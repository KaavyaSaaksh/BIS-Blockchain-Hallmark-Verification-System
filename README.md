## 🔐 BIS Blockchain Hallmark Verification System

A blockchain-based digital verification system designed to enhance gold hallmark authentication using:

- Blockchain ledger structure
- Status tracking (Active / Revoked / Suspicious)
- Fraud detection via scan intelligence
- Role-based access control (BIS Authority & Consumer)
- Immutable audit trail
